# values [![CI][ci-badge]][ci-link]

Create a list containing the values of an input object.

## Installation

Browser:

```sh
component install ndhoule/values
```

Node:

```sh
$ npm install values
```

## API

### `values(object : Object) => Array`

```javascript
values({ a: 1, b: 2, c: 3 });
//=> [1, 2, 3]
```
